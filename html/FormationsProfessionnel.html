Conseils pour s'intégrer dans un groupe :

Vous devez vous inscrire dans leur dynamique sociale. Chaque groupe a une dynamique qui est censée être un statu quo. Vous devez vous lancer dans leurs conversations comme si vous en faisiez partie depuis le début. S'il y a de l'harmonie, alors il est logique que vous fassiez partie du groupe.
